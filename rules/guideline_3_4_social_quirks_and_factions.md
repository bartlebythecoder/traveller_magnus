# Social Quirks, Factions, and Fringes (Topic 3.4)

## 1. Classic Traveller Logic
* **Faction Trigger**: Purely Referee-driven. The core rules do not use abstract faction strength or cultural quirk tables.
* **Balkanization (Gov 7)**: While RAW Book 6 suggests generating separate UWPs for individual territorial governments on Gov 7 worlds, **Project Scope** intentionally omits this hyper-granular splintering. The engine simply assigns the Gov 7 code to the world and defers faction details to the Referee.
* **Subordinate Governments**: Off-world settlements use the $1D$ formula. 
    * **Modifiers**: If Mainworld Gov is 6 (Captive Administration), the Subordinate Gov is automatically 6. If Mainworld Gov is 7+, apply $DM +2$.
* **Data Reference**: **Planetary Government Table** and **Subordinate Government Table**.

## 2. Mongoose 2nd Edition (WBH) Logic
* **Number of Factions**: $D3 + \text{Modifiers}$.
    * **Modifiers**: $+1$ (Gov 0/7), $-1$ (Gov A+).
* **Faction Identities**: Splinter groups (same Gov as world) vs. Dissident groups (different Gov).
* **Relative Strength**: Determined via a $2D$ roll on the **Faction Strength Table**.
* **Cultural Traits**: Narratively defined via the **Cultural Differences Table** (Xenophobia, Taboos, etc.).
* **Data Reference**: **Faction Strength Table** (Obscure to Overwhelming influence).

## 3. Traveller 5 (T5) Logic
* **The Cx Abstraction**: Social "chaos" is baked into the Cultural Extension string.
    * **Heterogeneity ($Pop + \text{Flux}$)**: Measures the spectrum of social behavior.
    * **Acceptance ($Pop + Ix$)**: Measures tolerance vs. xenophobia.
    * **Strangeness ($\text{Flux} + 5$)**: Measures deviation from interstellar norms.
* **Orbital Habitats**: Recognized as system assets (belts, gas giants) but lack a discrete factional generation formula in the core text.

## 4. RTT Engine Logic
* **Balkanization Logic**: Strictly gated by native Tech Level and Habitation type.
    * **Homeworld Rule**: $1D6 \le (\text{Native TL} - 9)$ results in a Balkanized (7) state. 
* **Colony/Outpost Stability**: Uses a flat $Population + 2D6 - 7$ for government with no secondary factional layers defined.
