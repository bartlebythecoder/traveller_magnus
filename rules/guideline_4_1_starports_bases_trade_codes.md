# Starports, Bases, and Trade Codes (Topic 4.1)

## 1. Classic Traveller (Book 6) Logic
* **Starport vs. Spaceport**: 
    * **Mainworld**: Generated via the **System Contents Table** (2D roll).
    * **Secondary Worlds**: Use **Spaceports** (1D roll). DMs: +2 (Pop 6+), -2 (Pop 1-).
* **Base Presence**: 
    * **Scout Base**: DMs: -1 (Port C), -2 (Port B), -3 (Port A). Forbidden for Ports E/X.
    * **Naval Base**: No DMs. Forbidden for Ports C, D, E, X.
* **Trade Codes**: Standard 13-code set (Ag, As, Ba, De, Fl, Hi, Ic, In, Lo, Na, Ni, Po, Ri, Wa) assigned strictly based on UWP triggers.
* **Data Reference**: **System Contents Table** (The primary engine for system-wide assets).

## 2. Mongoose 2nd Edition (WBH) Logic
* **System Anchor Rule**: Only the Mainworld possesses a **Starport** rating; all other bodies use **Spaceport** ratings.
* **Starport Formula**: $2D + \text{Population DMs} + \text{Strategic DMs}$ (Frontier -1, Trade Route +1).
* **Spaceport Formula**: $1D + \text{Population DMs}$.
* **Trade Codes**: Standard 18-code set. If a world meets all requirements for a code, it **must** gain that classification.
* **Data Reference**: **Trade Codes Table** (The 18-code Mgt2E standard).

## 3. Traveller 5 (T5) Logic
* **Importance (Ix) Coupling**: Starport quality is a primary input for world Importance (A/B = +1, D/E = -1).
* **Dynamic Tagging**: Trade classifications are split into **Physical Profile Tags** (HZ-dependent) and **Social Profile Tags** (Pop-dependent).
* **Expanded Codes**: Includes complex tags like **Pre-Agricultural (Pa)** based on specific Atm/Hyd/Pop 8-8-8 alignments.
* **Base Generation (Roll UNDER Target)**:
    * **Naval**: Port A (6-), Port B (5-).
    * **Scout**: Port A (4-), Port B (5-), Port C (6-), Port D (7-).
    * **Naval Depots**: Port A only, strict 1 in 1,000 frequency (0.1%).
* **Data Reference**: **Trade Classification Matrix** and **T5 Base Generation Tables**.

## 4. RTT Engine Logic
* **The Override Protocol**: Starport Class is normally a random roll ($2D + \text{Industry} - 7$), but is subject to **Fixed Overrides**:
    * **Outpost (Pop 0)**: Automatically Starport E (Nav-beacon/Cache).
    * **Industrial (Ind 5+)**: Minimum Starport E.
    * **Uninhabitable (Pop 1+)**: Minimum Starport E (Airlocks/Docking).
* **Constraint**: Starport quality is an emergent property of the world's **Habitation Type** and **Industry Level**.
